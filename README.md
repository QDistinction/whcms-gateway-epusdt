## 概述 ##
用于对接Epusdt的WHCMS第三方支付接口模块

## 目录结构 ##

```
whcms-gateway-epusdt/
 | epusdt.php
 │
 |-callback
 │      epusdt_notify.php
 │      epusdt_return.php
 │
 └─epusdt
     │  whmcs.json
     │
     └─lib
        epusdt.php
```

## 对接 ##

解压后文件复制到 `modules/gateways` 目录下即可
